Agent là gì?

Người đăng nhập vào phần mềm Subiz được gọi là agent. Agent có thể là người quản lý, nhân viên chăm sóc khách hàng, nhân viên kinh doanh và nhân viên Marketing.

Mỗi agent có email và mật khẩu riêng để đăng nhập Subiz làm việc.

Bạn quản lý được danh sách tất cả agent trên Subiz tại trang Cài đặt Agent

Nhiều agent có thể cùng tham gia trả lời khách hàng hoặc tự động chia riêng khách cho từng agent phụ trách.

Thêm agent vào Subiz

Một tài khoản Subiz có thể thêm nhiều agent vào cùng làm việc và phân chia quyền hạn riêng cho mỗi agent.

Hướng dẫn thêm agent mới:

  • Mở trang Cài đặt
  • Xem cột chức năng bên trái màn hình, tìm chức năng Agent
  • Chọn Mời agent
  • Nhập địa chỉ email của agent mới và chọn Phân quyền cho agent

  • Agent được mời sẽ nhận một email mời tham gia từ Subiz hoặc bạn gửi đường link cho agent tham gia vào Subiz.
  • Agent được mời sẽ tự nhập email và mật khẩu đăng nhập Subiz

Phân quyền Agent

Phân quyền agent là việc thiết lập các quyền hạn cụ thể cho từng agent trong phần mềm Subiz, giúp doanh nghiệp chuyên môn hóa công việc của từng nhân viên và nâng cao tính bảo mật thông tin.

Có 3 phân quyền chính trên Subiz là:

  • Chủ sở hữu: Tất cả quyền gồm cài đặt chức năng, thanh toán, quản lý toàn bộ khách, toàn bộ hội thoại, toàn bộ đơn hàng.
  • Quản lý: Bao gồm quyền cài đặt chức năng, quản lý toàn bộ khách, toàn bộ hội thoại, toàn bộ đơn hàng. Trừ quyền Thanh toán.
  • Agent: Chỉ có quyền xem khách hàng được giao, xem hội thoại được giao, xem đơn hàng được giao cho mình.

Đồng thời, bạn có thể chỉnh sửa riêng quyền sử dụng từng chức năng Subiz cho mỗi agent.

Trạng thái agent

Có 2 trạng thái agent:

  • Active: Agent được phép đăng nhập vào Subiz.
  • Inactive: Agent không được phép đăng nhập vào Subiz.

Hướng dẫn chỉnh sửa trạng thái agent:

  • Vào trang Cài đặt agent 
  • chọn Chỉnh sửa agent
  • Tìm mục Đổi trạng thái hoạt động - bật ON hoặc tắt OFF

Nhóm agent

Chức năng nhóm agent giúp bạn chia nhóm nhân viên để phân chia việc trả lời khách hàng. Ví dụ: nhóm miền Bắc, nhóm miền Nam, nhóm Kỹ thuật, nhóm CSKH,...

Hướng dẫn tạo nhóm agent:

  • Mở trang Cài đặt
  • Tìm chức năng Nhóm agent 
  • Chọn Tạo nhóm
  • Đặt tên nhóm và nhấp dấu + thêm thành viên agent
  • Lưu là xong

Xử lý tình huống thường gặp

CH1: Cách xử lý khi agent quên mật khẩu đăng nhập Subiz

Trả lời: Khi agent quên mật khẩu đăng nhập Subiz thì agent tự thay đổi được mật khẩu mới.

Cụ thể các bước như sau:

  • Đăng nhập Subiz trên máy tính, đường link là https://app.subiz.com.vn
  • Điền email đăng nhập Subiz, bấm tiếp tục
  • Nhấp nút Quên mật khẩu
  • Làm tiếp các bước xác nhận đổi mật khẩu

Gợi ý: Ngoài cách điền mật khẩu đăng nhập Subiz, agent có thể chọn cách Đăng nhập với Google. Subiz tự động liên kết với email Google mà bạn đã đăng nhập trên trên thiết bị của bạn.

CH2: Cách thay đổi email của agent

Trả lời: Bạn không thể thay đổi trực tiếp địa chỉ email của agent. Thay vào đó, bạn sẽ mời thêm agent có địa chỉ email mới.


Lưu ý: Bạn cần tắt OFF trạng thái hoạt động của agent cũ nếu agent không còn làm việc nữa.

CH 3: Cách thay đổi họ tên và agent đại diện của agent?

Có 2 cách để thay đổi họ tên và ảnh đại diện của agent.

Cách 1:

  • Mở Cài đặt agent
  • Chọn Chỉnh sửa
  • Nhấp vào ảnh đại diện để tải ảnh mới lên
  • Sửa Họ tên
  • Nhấp lưu cài đặt

Cách 2:

Mỗi agent tự chỉnh sửa thông tin cá nhân như sau:

  • Nhấp vào ảnh đại diện ở góc phải màn hình
  • Chọn Hồ sơ và tùy biến
  • Sửa các thông tin cần thiết
  • Nhấp Cập nhật là xong

CH4: Cách thay đổi email chủ sở hữu tài khoản

Trả lời: Khi công ty thay đổi hệ thống email hoặc thay đổi nhân sự, bạn sẽ tự thay đổi email chủ sở hữu tài khoản Subiz như sau:

  • Bước 1: Đăng nhập Subiz bằng email chủ sở hữu đang có
  • Bước 2: Mở trang Cài đặt agent
  • Bước 3: Chọn Mời agent, nhập vào địa chỉ email mới và chọn phân quyền Chủ sở hữu.
  • Bước 4: Thêm xong, bạn chọn Chỉnh sửa agent mới> Tìm mục Thay đổi chủ sở hữu và chọn Thay đổi
  • Lưu cài đặt là xong.

Sau đó, bạn đăng nhập Subiz và quản lý tài khoản bằng email mới.

CH5: Cách xóa agent trên Subiz

  • Mở trang Cài đặt
  • Tìm chức năng Agent
  • Chọn chỉnh sửa agent
  • Tìm mục Xóa khỏi doanh nghiệp
  • Lưu cài đặt là xong

CH6: Cách cài đặt agent giám sát viên

Trả lời: Agent giám sát viên được nhận và tham gia trả lời tất cả hội thoại của khách hàng. Bạn cài đặt agent giám sát viên tại trang Cài đặt Rule phân phối.

 

CH7: Tại sao agent mới không xem được tin nhắn cũ của khách?

Trả lời: Trên trang Tin nhắn, Agent mới chỉ nhận được các hội thoại mới của khách từ thời điểm agent được thêm vào Subiz.

Khi khách có quay lại nhắn tin thì agent mới sẽ nhận được tin nhắn của khách và xem được lịch sử cũ.

Agent mới muốn xem báo cáo các khách cũ nhắn tin, Agent mới thực hiện như sau:

  • Mở trang Thống kê
  • Chọn Thống kê hội thoại
  • Chọn Danh sách hội thoại và xem chi tiết.

CH8: Cách cài đặt agent nhận chat và trả lời khách hàng

Trả lời: Bạn sẽ cài đặt chức năng Rule phân phối để tự động lọc hội thoại của khách và giao cho agent trả lời chăm sóc khách. Xem hướng dẫn cài đặt Rule phân phối..

CH9: Hướng dẫn agent đổi ngôn ngữ trên trang app Subiz

Trả lời. Mỗi agent có thể tự chọn ngôn ngữ trang làm việc Subiz là Tiếng Anh hoặc Tiếng Việt.

  • Trên trang Subiz, nhấp vào ảnh đại diện ở góc phải màn hình
  • Chọn Hồ sơ và tùy biến
  • Chọn Thông tin cá nhân
  • Thay đổi mục Ngôn ngữ

CH10. Cách đăng xuất tài khoản Subiz

Agent nhấp vào ảnh đại diện ở góc phải màn hình, chọn nút Đăng xuất.

CH11: Cách xử lý khi agent nhận thông báo “Bạn không có quyền xem khách này?”

Agent có thông báo “Bạn không có quyền xem khách này” có nghĩa agent chỉ được phân quyền chỉ xem khách được giao, không được xem khách của đồng nghiệp.

Agent cần trao đổi lại cấp trên để thay đổi phân quyền cho mình được xem toàn bộ khách nếu cần. Xem thêm cài đặt phân quyền agent.
