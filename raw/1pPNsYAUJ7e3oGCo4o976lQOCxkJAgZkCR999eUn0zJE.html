Tích hợp Website trên Subiz

Kênh Website hoạt động như thế nào?

Khi khách hàng truy cập website, cửa sổ chat Subiz ở góc màn hình hiện ra, cho phép người dùng liên hệ tới tư vấn viên ngay tức thì bằng những tin nhắn trực tiếp.

Chỉnh sửa thiết kế cửa sổ phù hợp với website và nhóm khách hàng mục tiêu kết hợp nút chat, bot chào khách, tự động gửi tin nhắn quảng cáo, hỏi thông tin khách hàng trước khi chat,... là một cách để tạo ấn tượng và tăng tương tác với khách hàng.

Hướng dẫn tích hợp Website trên Subiz

  • Đăng nhập Subiz qua đường link https://app.subiz.com.vn
  • Mở trang Cài đặt (Biểu tượng bánh răng góc màn hình trên bên phải)
  • Nhấp chuột chọn Website ở mục “TÍCH HỢP”
  • Chọn nút Cài lên website ở góc phải màn hình

Tiếp tục điền thêm tên miền website để biết bạn cài đặt cửa sổ chat trên website nào. Ví dụ: Subiz.com.vn.

  • Sao chép mã nhúng Subiz và dán vào code website trong thẻ body.
  • Kiểm tra cài đặt thành công chưa? Vào website, F5 tải trang để cập nhật thay đổi và thấy cửa sổ chat ở góc phải màn hình là cài đặt thành công.

Cách cài đặt Subiz trên Website

Subiz và Wordpress

  • Bước 1: Lấy ID tài khoản

Đăng nhập app.subiz.com.vn bằng tài khoản đã đăng ký, vào Cài đặt> Copy ID tài khoản

Bước 2: Cài đặt Plugin Subiz

Đăng nhập tài khoản WordPress > Mở Plugins > Chọn Add new Plugin>Gõ chữ Subiz > click chuột tới Install and activate> Manager Plugin.

  • Bước 3: Thêm cửa sổ Chat vào Subiz

Sau khi cài đặt và kích hoạt plugin Subiz > chọn “Subiz” trên thanh công cụ> nhập ID tài khoản đã copy ở bước 1> Cài đặt.

  • Bước 4: Truy cập vào website và xem cửa sổ chat của bạn

Để kiểm tra Subiz cài đặt vào Website thành công hay chưa, bạn sẽ truy cập vào trang Website, nếu thấy cửa sổ chat Subiz hiển thị ở góc màn hình tức đã cài đặt thành công.

Subiz trên Ladipage

Bước 1: Đăng nhập tài khoản Ladipage và chọn Landing page bạn muốn đặt Subiz.

Bước 2: Trên Ladipage, bạn vào Tích hợp>Mã Tracking Global>Tạo mã ngay.

Lúc này sẽ có một bản thông tin hiển thị bạn tiếp tục chọn: Pixel cá nhân> kéo chuột xuống dưới cùng vào Clik Trước thẻ </body> rồi dán mã nhúng Subiz. Sau đó Save.

(Lưu ý: Lấy mã nhúng Subiz ở bước tích hợp Website)

Bước 4: Bạn xuất bản, và truy cập vào Landing Page để xem sự kết hợp sửa Subiz và Ladipage. Ngoài ra, bạn cũng có thể tùy chỉnh màu, button,ngôn ngữ cho cửa sổ Subiz này ngay tại Ladipage.

Subiz trên website Sapo

Bước 1: Đăng nhập trang quản trị website của Sapo > Chọn Website.

Bước 2: Chọn Giao diện > Thao tác > Chỉnh sửa HTML/CSS.

Bước 3: Chọn Theme.bwt > Tìm thẻ “body” > Sao chép mã nhúng Subiz và dán trước thẻ  “body” > Lưu để hoàn thành.

Subiz trên website Haravan

Bước 1: Đăng nhập trang quản lý website của Haravan

Bước 2: Chọn Website > chọn Giao diện > chọn

Bước 3: Chọn theme.liquid trong Layouts > Tìm thẻ “head” > Sao chép mã nhúng Subiz và dán ngay sau thẻ “head” > chọn Lưu để hoàn thành

Tình huống thường gặp

Sau tích hợp, cửa sổ chat không hiển thị trên Website do đâu và xử lý thế nào?

Cửa sổ chat Subiz không hiện trên website, thường có 4 nguyên nhân chính:

Nguyên nhân 1: Chưa lưu thành công mã nhúng Subiz trong code web

  • Cách kiểm tra: Mở tab trình duyệt mới rồi mở lại website. Chọn Ctrl U, rồi Ctrl F và nhập tìm Subiz. Có tìm được mã nhúng không? Nếu không, thì do chưa lưu thành công sau khi dán mã nhúng Subiz vào code web.
  • Cách xử lý: Quay lại trang admin code web tìm xem có sử dụng plugin cache nào không? Nếu có xóa cache để lưu thành công cài đặt Subiz.

Nguyên nhân 2: Sai mã nhúng khi đặt vào code website

  • Kiểm tra: So sánh mã nhúng gốc của Subiz sao chép trên trang Cài lên website và mã nhúng đang đặt trong code web tại trang https://www.diffnow.com/ này.
  • Cách xử lý: Sao chép lại mã nhúng Subiz đúng và dán vào code website. Nếu mã nhúng Subiz vẫn bị thay đổi khác hãy sử dụng đoạn mã nhúng dưới đây:

<script>

  window._sbzaccid = 'acpzooihzhalzeskamky'

  window.subiz = function () {

    window.subiz.q.push(arguments)

  }

  window.subiz.q = []

  window.subiz('setAccount', window._sbzaccid)

</script>

<script src="https://widget.subiz.net/sbz/app.js?account_id=acpzooihzhalzeskamky"></script>

Lưu ý: Đổi mã tài khoản Subiz của bạn vào đoạn mã trên (sbzaccid = 'acpzooihzhalzeskamky', account_id=acpzooihzhalzeskamky), Xem mã tài khoản Subiz của bạn.

Nguyên nhân 3: Sai vị trí mã nhúng Subiz trong code website. 

  • Kiểm tra: Xem mã nhúng Subiz đã đặt đúng trong thẻ “body” chưa? Trong code web sẽ có thẻ “body” mở và thẻ “body” đóng.
  • Cách xử lý: Chuyển mã nhúng từ vị trí sai  vào vị trí trong thẻ “body” của code web.

Nguyên nhân 4: Có nhiều mã nhúng Subiz khác nhau trong code web.

  • Kiểm tra: Mở tab trình duyệt mới. Mở website lên, chọn Ctrl U, chọn Ctrl F và điền Subiz.  Hiện ra từ 2 mã nhúng Subiz trở lên trong code web.
  • Cách xử lý: Giữ 1 mã nhúng đúng đang đặt trong thẻ “body” và xóa tất cả mã nhúng Subiz còn lại.

Dùng Google Tag Manager để cài đặt Subiz lên website được không?

Có. Bạn cài đặt Subiz qua Google Tag Manager được.

Khi bạn sử dụng Google Tag Manager trên website, có thể tích hợp Subiz chat vào website thông qua Google Tag Manager.

Bước 1: Đăng nhập Tagmanager.google.com

Bước 2: Chọn account quản lý website sẽ tích hợp Subiz chat tại All account.

Bước 3: Tạo Tag mới Tại WORKSPACE chọn Tags, nhấp NEW

Bước 4: Untitled Tag: Điền tên Tag để phân biệt các Tag và chức năng của Tag

Bước 5: Tag Configuration: Chọn Custom HTML, Sao chép mã nhúng Subiz và dán vào ô HTML

Chọn (tick) Support document.write

Bước 5: Triggering: Chọn All Pages nếu muốn hiển thị cửa sổ chat Subiz trên tất cả các trang của website

Lưu ý: Trigger là điều kiện bạn muốn thẻ Tag Subiz chat sẽ hoạt động. Bạn có thể tùy chọn Trigger theo nhu cầu riêng. Ví dụ:

  • Tất cả các trang của website (All Pages) sẽ hiển thị cửa sổ Subiz chat.
  • Chỉ một số trang (Some Pages) sẽ hiển thị cửa sổ Subiz chat.
  • Khách có một hành động cụ thể như Click, kéo scroll chuột, hay sau một khoảng thời gian vào website,.... sẽ hiển thị cửa sổ Subiz chat.

Bước 6: Chọn SAVE, chọn SUBMIT ở góc phải màn hình > Điền thông tin mô tả trong mục Publish and Create Version > chọn PUBLISH để hoàn thành.

Bước 7: Đăng nhập vào website và F5 để tải lại trang, kiểm tra cửa sổ chat Subiz đã hiển thị trên website.
