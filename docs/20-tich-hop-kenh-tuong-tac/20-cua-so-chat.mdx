# Cửa sổ chat




<b>Cửa sổ chat</b> là một thành phần của website & cũng là một trong những hình ảnh đại diện cho Doanh nghiệp. Bạn có thể tùy chỉnh các thông tin trên cửa sổ chat sao cho hòa hợp với hình ảnh chuyên nghiệp của website.



Tại Subiz, bạn hoàn toàn có thể custom hiển thị cửa sổ chat riêng cho mỗi website Doanh nghiệp. Từ màu sắc, tên hiển thị, hình ảnh đại diện cho tới nội dung xuất hiện trên cửa sổ chat.

### Cách thay đổi ảnh nút chat Subiz {#cach-thay-doi-anh-nut-chat-subiz}


Trên giao diện máy tính, nút chat kêu gọi khách nhắn tin có thể thay đổi là ảnh nhỏ hoặc ảnh lớn nhằm thu hút sự chú ý của khách hàng. Hướng dẫn thay đổi nút chat: 

- Nhấp <b>Nút chat </b>
- Click vào ảnh nút chat để thay đổi, bạn có thể tùy chọn ảnh nút chat có sẵn trong thư viện Subiz hoặc tải ảnh bạn thiết kế lên.




![](https://vcdn.subiz-cdn.com/file/fismoignuiadzuxjyron_acpxkgumifuoofoosble/17de751a7930c0dc1043f4da7c350d4c.png)


Trên điện thoại, nếu thấy Nút chat hiển thị lớn quá, bạn hoàn toàn có thể <b>lựa chọn icon nhỏ hơn</b> để phù hợp với kích thước nhỏ. Các bước như sau:

01. Vào Cài đặt <b>Chỉnh sửa cửa sổ chat</b>
11. Chọn <b>Nút chat</b>, sau đó ở Nút chat trên điện thoại di chuột vào hình ảnh nhấn <b>Thay đổi</b>
21. Chọn hình ảnh hiển thị nút chat phù hợp rồi <b>Lưu thay đổi</b>.

Để thực hiện chỉnh sửa thiết kế cửa sổ chat riêng cho từng website, bạn làm theo các bước sau:

### Thiết kế cửa sổ chat riêng cho landing page {#thiet-ke-cua-so-chat-rieng-cho-landing-page}


Không chỉ hỗ trợ hiển thị cửa sổ chat đẹp custom riêng cho mỗi website, Subiz còn hỗ trợ khách hàng tạo cửa sổ chat với từng link con trên trang web, ví dụ như:

- Link danh mục sản phẩm
- Link con từng dịch vụ
- Landing page

Bạn có thể cài đặt nút CTA (kêu gọi khách hàng động) ở vị trí bất kỳ trên website và mở cửa sổ chat được thiết kế theo nội dung từng trang khi khách nhấp chuột vào.



<b>Hướng dẫn cài đặt:</b>

- Tắt OFF nút chat Subiz tại trang [Cài đặt cửa sổ chat](https://app.subiz.com.vn/chatbox/design) (nếu cần)
- Đặt mã API mở cửa sổ chat dưới đây vào code website, để ra lệnh mở cửa sổ chat khi khách click vào hình ảnh hoặc đường link trên website.

### Yêu cầu khách điền form thông tin trước khi chat {#yeu-cau-khach-dien-form-thong-tin-truoc-khi-chat}


Để phục vụ cho việc thu thập thông tin khách hàng, bạn cũng có thể tạo form yêu cầu điền các trường thông tin trước khi bắt đầu hội thoại qua Subiz. 

Cài đặt rất đơn giản, theo các bước sau:



- Mở Chỉnh sửa cửa sổ chat
- Nhấp chọn <b>Hỏi thông tin trước khi chat</b>
- Bật ON và chọn hỏi thông tin cần thiết của khách (Họ tên, email, số điện thoại…)
- Cập nhật là xong


![](https://vcdn.subiz-cdn.com/file/fismoigsmsjvvvsdzbam_acpxkgumifuoofoosble/6efd2368b426099d9b968e4237651fac.png)




<b>Lưu ý</b>: Bạn muốn hỏi nhiều thông tin khác theo đặc thù riêng của Doanh nghiệp, bạn sẽ cần tạo trường thông tin mới tại trang <b>[Cài đặt Trường thông tin](https://app.subiz.com.vn/settings/user-attributes)</b>, rồi quay lại thêm hỏi thông tin vào bảng hỏi.
