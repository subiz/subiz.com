# Nhãn phân loại

## Nhãn phân loại là gì? {#nhan-phan-loai-la-gi}


<b>Nhãn phân loại khách hàng</b> là một trường thông tin của khách hàng có định dạng nút. Nhãn phân loại sẽ giúp bạn:

- Tổng hợp thông tin khách hàng
- Phân loại các nhóm khách hàng mục tiêu
- Thực hiện các chiến dịch chăm sóc, bán hàng cho khách hàng mục tiêu

## Tạo nhãn phân loại mới {#tao-nhan-phan-loai-moi}


Hướng dẫn tạo mới nhãn phân loại khách hàng:

- Tới trang Cài đặt & chọn <b>[Nhãn phân loại](https://app.subiz.com.vn/settings/label)</b>
- Nhấn <b>Tạo nhãn</b> ở góc trên bên phải màn hình
- Nhập thông tin nhãn phân loại
- Bấm <b>Lưu</b> để hoàn thành.
