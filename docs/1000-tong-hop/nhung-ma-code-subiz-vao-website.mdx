# Nhúng mã code Subiz vào website

Sau khi [lấy được mã code cửa sổ chat Subiz](https://subiz.com.vn/docs/583253150-cua-so-chat#lay-ma-nhung), bạn cần đi tới trang quản trị website và làm theo hướng dẫn tương ứng với nền tảng của doanh nghiệp:

- WordPress
- LadiPage
- Sapo
- Haravan
- Google Tag Manager

## WordPress {#wordpress}

### Tải plugin {#tai-plugin}

- Đăng nhập vào trang Quản trị WordPress, tại menu bên trái chọn <b>Plugins</b>, nhấn <b>Add New Plugin</b>
- Tại ô tìm kiếm plugin, nhập từ khóa “<i>Subiz”</i>
- Chọn plugin<b> Subiz - Live chat</b>, nhấn <b>Cài đặt ngay</b>, nhấn <b>Kích hoạt</b> và chờ vài giây.

![](https://vcdn.subiz-cdn.com/file/fisshsxiecryklnbxqlc_acpxkgumifuoofoosble/f3bbac0452a8366d4f6a8ddffce2b329.png)

### Kết nối với tài khoản Subiz {#ket-noi-voi-tai-khoan-subiz}

Sau khi thao tác kích hoạt thành công, bạn sẽ được đưa về màn hình danh sách plugin cài trên website. Lúc này hãy tiếp tục thao tác như sau:

- Bấm <b>Cài đặt</b> ngay phía dưới tên Subiz
- Nhấn nút <b>Bắt đầu</b> và điền [ID tài khoản Subiz](https://app.subiz.com.vn/settings/) của bạn vào ô tương ứng
- Nhấn <b>Cài đặt</b> để hoàn tất.

### Đổi tài khoản Subiz {#doi-tai-khoan-subiz}

Trong trường hợp có thay đổi về thông tin tài khoản Subiz nhận tin nhắn cũng như quản lý cửa sổ chat. Bạn chỉ cần thao tác đổi tài khoản đơn giản với các bước sau:

- Vào phần cài đặt plugin Subiz
- Nhấn <b>Đổi tài khoản</b>
- Điền account ID mới rồi bấm <b>Cập nhật</b>.

## LadiPage {#ladipage}

1. [Đăng nhập tài khoản](https://ladipage.vn)[LadiPage](https://ladipage.vn) và chọn Landing page muốn đặt Subiz
2. Trên thanh công cụ nằm ở góc trên màn hình, nhấp vào <b>Thiết lập</b>.
3. Chọn mục <b>Mã JavaScript/CSS</b>.
4. Dán [mã nhúng Subiz](https://subiz.com.vn/docs/583253150-cua-so-chat#lay-ma-nhung) vào trước thẻ <b>&lt;/body&gt;</b> trong phần thiết lập trang.
5. Nhấn <b>Lưu</b> &gt; Chọn <b>Xuất bản</b>.

![](https://vcdn.subiz-cdn.com/file/fisrloeofspatwpzluzq_acpxkgumifuoofoosble/b6c50b7096014b53d770c8f1e0d78925.png)

## Sapo {#sapo}

1. Đăng nhập trang quản trị website của Sapo
2. Chọn Website &gt; Chọn <b>Giao diện</b>
3. Tại giao diện, nhấn vào phần <b>Thao tác</b> và chọn <b>Chỉnh sửa HTML/CSS</b>.
4. Tìm file <b>Theme.bwt</b> ở thanh menu bên trái và nhấn mở file
5. Tìm thẻ “body” &gt;[https://app.subiz.com/settings/install](https://app.subiz.com/settings/install)dán [mã nhúng Subiz](https://subiz.com.vn/docs/583253150-cua-so-chat#lay-ma-nhung) ngay trên thẻ “body”
6. <b>Lưu</b> để hoàn thành.

![](https://vcdn.subiz-cdn.com/file/fisgyrdigqwulvcvrvfb_acpxkgumifuoofoosble/unnamed.gif)

## Haravan {#haravan}

1. Đăng nhập [trang quản lý website của Haravan](https://myharavan.com/admin/auth/login)
2. Chọn <b>Website</b> &gt; chọn <b>Giao diện</b>
3. Chọn giao diện website muốn chèn mã và nhấp chọn <b>Chỉnh sửa code</b>.

![](https://vcdn.subiz-cdn.com/file/fisrloeokcglnjxhvtfi_acpxkgumifuoofoosble/90b091d38880e7d50ed0c376ce9669f3.png)

1. Tìm file <b>theme.liquid</b> ở thanh menu bên trái và nhấn mở file

![](https://vcdn.subiz-cdn.com/file/fisrloeolurintuynmxm_acpxkgumifuoofoosble/23f07769091d4f0592da52738f8a1bbd.png)

1. Dán [mã nhúng Subiz](https://subiz.com.vn/docs/583253150-cua-so-chat#lay-ma-nhung) ngay dưới thẻ <b>“&lt;/head&gt;”</b>
2. Chọn <b>Lưu</b> để hoàn thành.

## Google Tag Manager {#google-tag-manager}

Khi website của bạn đang sử dụng Google Tag Manager \(GTM\), bạn có thể nhúng mã Subiz Chat thông qua Google Tag Manager thay vì chèn trực tiếp vào codewebsite. Thực hiện như sau:

Bước 1: Đăng nhập [Tagmanager.google.com](https://tagmanager.google.com/#/home)

Bước 2: Chọn account quản lý website sẽ tích hợp Subiz chat tại <b>All account</b>.

Bước 3: Tại <b>WORKSPACE </b>chọn<b> Tags, </b>nhấp<b> NEW</b> để tạo tag mới

Bước 4: Điền tên Tag để phân biệt các Tag và chức năng của Tag

Bước 5: Nhúng mã code Subiz

- Tại <b>Tag Configuration &gt; </b>Custom HTML &gt; dán [mã nhúng Subiz](https://subiz.com.vn/docs/583253150-cua-so-chat#lay-ma-nhung)
- Chọn ✓<b>Support document.write</b>
- Chọn <b>All Pages</b> ở <b>Triggering</b> nếu muốn hiển thị cửa sổ chat Subiz trên tất cả các trang của website

  <b>Lưu ý</b>: Trigger là điều kiện bạn muốn thẻ Tag Subiz chat sẽ hoạt động. Bạn có thể tùy chọn Trigger theo nhu cầu riêng. Ví dụ:

- Tất cả các trang của website \(All Pages\) sẽ hiển thị cửa sổ Subiz chat.
- Chỉ một số trang \(Some Pages\) sẽ hiển thị cửa sổ Subiz chat.
- Khách có một hành động cụ thể như Click, kéo scroll chuột, hay sau một khoảng thời gian vào website,.... sẽ hiển thị cửa sổ Subiz chat.

Bước 6: Chọn <b>SAVE</b>, chọn <b>SUBMIT</b> ở góc phải màn hình

Bước 7: Điền thông tin mô tả trong mục <b>Publish and Create Version</b> &gt; chọn <b>PUBLISH</b> để hoàn thành.

![](https://vcdn.subiz-cdn.com/file/fismoidztllutekcxcwa_acpxkgumifuoofoosble/11cb633c561f224ba44dfb6a9e320599.png)

## Kiểm tra hoạt động cửa sổ chat Subiz {#kiem-tra-hoat-dong-cua-so-chat-subiz}

Vào website, F5 tải trang để cập nhật thay đổi và thấy cửa sổ chat ở góc phải màn hình là cài đặt thành công. Nhắn tin test trên cửa sổ chat và quay lại [trang quản lý hội thoại chat trên Subiz](https://app.subiz.com.vn/convo) để phản hồi tin nhắn đó.

Tiếp theo, cùng [thiết kế hiển thị, lời chào cửa sổ chat](https://subiz.com.vn/docs/583253150-cua-so-chat#chinh-sua) theo thông tin của doanh nghiệp mình nhé.

## Kiểm tra {#kiem-tra}

## Gỡ Subiz khỏi website {#go-subiz-khoi-website}

Để gỡ Subiz khỏi website, bạn cần xóa mã nhúng Subiz khỏi trang quản trị website. Cách thực hiện tương tự như khi đặt mã nhúng Subiz vào website trước đây.

Nếu không rành kỹ thuật, bạn hãy nhờ kỹ thuật website hoặc kỹ thuật công ty bạn thực hiện giúp.
