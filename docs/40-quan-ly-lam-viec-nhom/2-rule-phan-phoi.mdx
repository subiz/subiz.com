# Rule phân phối

## Rule phân phối là gì? {#rule-phan-phoi-la-gi}




<b>[Rule phân phối](https://app.subiz.com.vn/settings/rule-setting)</b> là chức năng tự động phân chia hội thoại của khách hàng rồi giao cho nhân viên trả lời tư vấn khách, giúp chuyên môn hóa công việc cho từng nhân viên và nâng cao chất lượng dịch vụ khách hàng.



Ví dụ: 

- Rule chia theo vùng miền: Chia khách miền Bắc cho nhân viên miền Bắc, chia khách miền Nam cho nhân viên miền Nam.
- Rule theo kênh tương tác: Chia khách từ Fanpage cho nhân viên A, chia khách website cho nhân viên B.
- Rule theo hiệu suất của nhân viên: Chia đều khách cho mỗi nhân viên, chia khách cho nhân viên trả lời nhanh nhất.


![](https://vcdn.subiz-cdn.com/file/fismpqracmkvyonxyebr_acpxkgumifuoofoosble/6aaca671e1b8f14aa6f4ababeb42b8d8.png)




Một số điểm nổi bật của Rule phân phối:

- Bạn được tạo nhiều rule phân phối khác nhau
- Rule phân phối hoạt động như phễu lọc gồm nhiều tầng lớp. Khách liên hệ được lọc theo thứ tự rule từ trên xuống dưới.

## Hướng dẫn tạo rule phân phối {#huong-dan-tao-rule-phan-phoi}


Để tạo rule phân phối bạn cần thức hiện 2 bước:



<b>Bước 1: Xác định quy tắc phân chia khách cho agent</b>

- Đặc điểm phân loại của khách hàng là gì? Ví dụ theo vùng miền, theo nguồn khách hàng, theo thời gian khách liên hệ,....
- Agent nào phụ trách trả lời khách hàng?



Khi đó, số lượng rule cần tạo sẽ tương ứng với số lượng nhóm khách của bạn



<b>Bước 2: Cài đặt rule phân phối</b>

Hướng dẫn cài đặt:

- Mở trang Cài đặt
- Xem cột chức năng bên trái, tìm [Rule phân phối](https://app.subiz.com.vn/settings/rule-setting)
- chọn Tạo Rule phân phối
- Nhập tên Rule để nhận biết rule này làm nhiệm vụ gì?
- Chọn điều kiện lọc khách hàng
- Chọn Phân phối tới, thêm agent trả lời khách
- Nhấp Tạo Rule là xong

## Rule phân phối mẫu {#rule-phan-phoi-mau}

### Rule phân phối theo kênh Website, Fanpage, Zalo {#rule-phan-phoi-theo-kenh-website-fanpage-zalo}




<b>Tình huống</b>: Doanh nghiệp có 3 kênh chat khách hàng là website, Fanpage, Zalo. Yêu cầu agent 1 trực chat trên website, agent 2 trực chat trên Fanpage và Zalo



<b>Giải pháp</b>: Tạo 2 Rule phân phối tương ứng 2 nhóm chat website và chat Fanpage + Zalo

- <b>Rule 1</b>: Điều kiện Rule chọn kênh tương tác là website - Chọn Phân phối 1 agent cụ thể
- <b>Rule 2</b>: Điều kiện Rule chọn kênh tương tác là Messenger + Bình luận Fanpage + Zalo - Chọn Phân phối tới 1 agent cụ thể


### Rule phân phối theo vùng miền {#rule-phan-phoi-theo-vung-mien}




<b>Tình huống</b>: Doanh nghiệp có 2 chi nhánh Hà Nội và Hồ Chí Minh. Yêu cầu nhóm agent Hà Nội trực chat khách từ các tỉnh thành lân cận Hà Nội và nhóm HCM trực chat khách từ các tỉnh thành lân cận HCM



<b>Giải pháp</b>: Tạo nhóm agent HN và nhóm Agent HCM; Sau đó tạo 2 rule phân phối tương ứng nhóm khách chat vùng HN và nhóm khách chat vùng HCM

- <b>Bước 1</b>: Tạo nhóm agent tại [trang Nhóm agent](https://app.subiz.com.vn/settings/agents-group)
- <b>Bước 2</b> : Tạo rule phân phối cho nhóm agent tương ứng theo vị trí địa lý

- <b>Rule 1</b>: Điều kiện Rule chọn Vị trí thành phố là Hanoi và thêm các thành phố lân cận - Phân phối tới nhóm agent HN
- <b>Rule 2</b>: Điều kiện Rule chọn Vị trí thành phố là Ho Chi Minh City và thêm tỉnh/thành lân cận - Phân phối tới nhóm agent HCM


### Rule phân phối theo thời gian làm việc {#rule-phan-phoi-theo-thoi-gian-lam-viec}


<b>Tình huống</b>: Trong giờ làm việc, cuộc chat sẽ chia đều vòng tròn cho agent. Ngoài giờ làm việc, các cuộc chat chia cho agent trả lời nhanh nhất để khi agent trả lời thì các agent khác không xem chat đó nữa.



<b>Giải pháp</b>: Tạo 2 rule phân phối tương ứng chat trong trong giờ làm việc và chat ngoài giờ làm việc

- <b>Rule 1</b>: Điều kiện Rule chọn Thời gian Trong giờ làm việc - Phân phối đồng đều theo vòng
- <b>Rule 2</b>: Điều kiện Rule chọn Thời gian Ngoài giờ làm việc - Phân phối tới agent trả lời nhanh nhất
